An http proxy server (with local cache support) for psvita/ps3/psp3. It supports file resuming which ps3 proxy server doesn't.
Based on Tiny HTTP Proxy in Python (Author SUZUKI Hisao)

format in cache.txt:
replace_url->replace_file_path
search:replace_filename->replace_file_path
re:replace_url_regular_expression->replace_file_path
